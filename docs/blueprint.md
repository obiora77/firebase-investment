# **App Name**: Olowo Investments

## Core Features:

- User Registration & Authentication: Secure registration and login system with email verification and password reset functionality.
- Investment Plans Display: Display a list of investment plans with their respective details such as investment range, profit percentage, and duration.
- User Dashboard: Provide a user dashboard with a summary of their financial status, including total balance, active investment amount, total profit, and recent transactions.
- Deposit and Investment Functionality: Enable users to deposit funds using various methods and invest in available plans, with automated profit calculation based on plan parameters. A tool for adjusting user balances to account for successful payment
- Withdrawal Request System: Allow users to request withdrawals from their available balance, with a pending state for admin approval.
- Admin Panel: Provide an admin panel for managing users, investments, finances, and support tickets.
- Real-time Crypto Price Ticker: Display a real-time ticker of cryptocurrency prices using CoinGecko API.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to evoke trust and security.
- Background color: Light gray (#F5F5F5) for a clean, professional look.
- Accent color: Gold (#FFD700) to highlight important information and CTAs.
- Body font: 'Inter' sans-serif for a modern and readable experience.
- Headline font: 'Space Grotesk' sans-serif for impactful headings and titles; Pair headlines with 'Inter' for body text.
- Use professional, clear icons for navigation and representing different investment aspects.
- Maintain a clean, well-organized layout with clear visual hierarchy to guide users through the platform.
- Use subtle animations and transitions to enhance user experience and provide feedback.
